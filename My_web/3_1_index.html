var, let, const의 차이

  1. var

    변수의 재선언, 재할당이 가능하다.

  2. let

    변수의 재선언 불가능하고, 재할당이 가능하다.

  3. const

    변수의 재선언, 재할당 모두 불가능하다.

  4. 회고

    HTML, CSS, JavaScript를 이해할 수 있었다. 또한 객체와 메소드 등 평소 알고 있었지만 와닿지 않았었던 용어들을 이해할 수 있어서 좋았다. 매번 헷갈려서 검색했었는데 직관적으로 이해할 수 있었다. 전반적인 강의는 쉬운 편이었고 흥미가 있었다. 그리고 웹을 구현하면서 데이터 및 인공지능과 관련해서 연결 시킬 수 있겠다는 희망이 생겼다.
