어제 처음으로 팀플을 했었던 팀원분들을 만나고 즐거운 시간을 보내서 좋았다!
그러나 문제는 이번주 과제이다…. 만만치 않았다. 로또까지는 할 수 있는데, 타이머와 시계는 조금 무리인 것 같다….. CSS가 가장 문제이다.
가운데 정렬이 되어야 하는데, 무슨 문제가 있는 것인지 계속해서 안된다. CSS, HTML, JavaScript 세 가지를 모두 다루는 게 전부 알지 못해서 분한 마음이 든다. 특히 타이머 기록이 가장 신경이 쓰였다.
문자열로 바꾸어서 버튼을 누르면 기록하는 형식으로 만들려고 했지만 쉽지 않았고 문자열로 바꾸고 나서 css로 나타내는 것이 어렵다.
시계 초침은 내가 제안한 것이긴 한데 포기하기로 했다. 아무래도 전혀 모르겠다. 로또는 성공했지만 css 배열이 문제이고 시간 기록은 아이디어는 있는데 실행 못하고 아날로그 시계는 전혀 모르겠다.
